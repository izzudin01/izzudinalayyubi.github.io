IoT Watering System

Simple automation prototype: read sensor condition → apply control rules → drive actuator (pump/valve) for watering.

GitHub Repo

Objective

  • Automate watering based on measured conditions (e.g., soil/moisture logic).
  • Reduce manual work and avoid over/under-watering.
  • Keep logic simple, testable, and safe for hardware.

System Overview

  • Sensing: reads environmental/soil condition.
  • Decision: threshold + timing rules (basic control).
  • Actuation: pump/valve control via relay/driver.

My Contribution

  • Implemented control rules with safe actuator handling (debounce/timing to avoid rapid switching).
  • Integrated sensors + actuator wiring concept for a working prototype.
  • Documented usage assumptions and test scenarios.

Tools & Tech

IoT Microcontroller Sensors Relay/Driver Control Logic

Operations Angle

  • Demonstrates basic automation loop (sense → decide → act).
  • Shows reliability thinking (timers, stability, safe switching).
  • Transferable mindset to industrial automation problems.
