Air Quality Monitoring (IoT)

IoT monitoring project to observe air quality indicators over time and present them in a simple dashboard/log.

GitHub Repo

Objective

  • Collect air quality-related sensor readings and track trends.
  • Store/display data in a readable format for users.
  • Enable basic alerting/threshold logic (optional).

System Overview

  • Microcontroller reads environmental sensors (air quality + basic environment metrics).
  • Data is logged and shown via simple UI/dashboard approach.
  • Designed as a practical monitoring prototype.

My Contribution

  • Integrated sensor acquisition + basic data processing/cleaning logic.
  • Built a simple visualization/logging flow to make readings usable.
  • Structured the project as an end-to-end IoT prototype (sense → process → present).

Tools & Tech

IoT Microcontroller Sensors HTML/CSS/JS (optional UI) Data Logging

Operations Angle

  • Monitoring makes problems visible (baseline + anomaly awareness).
  • Supports safety and health awareness in indoor environments.
  • Good foundation for preventive actions (ventilation checks, housekeeping).
