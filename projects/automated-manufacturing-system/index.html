Automated Manufacturing System (Factory I/O + PLC)

3D simulation of a manufacturing line with PLC-based control logic: sequencing, interlocks, and process flow stability.

Objective

  • Simulate a production flow (machining + assembly stations).
  • Implement PLC logic for sequencing and coordination.
  • Validate interlocks and safe transitions between states.

My Contribution

  • Designed sequence logic and state transitions.
  • Programmed Modicon M221 PLC using ladder logic and structured text.
  • Debugged timing/handshake issues in simulation and improved robustness.

Key Concepts Demonstrated

  • System-level thinking: stations, buffers, and dependencies.
  • Interlocks: prevent unsafe or invalid operations.
  • Reliability mindset: fault handling and deterministic sequences.

Tools & Tech

Factory I/O PLC Modicon M221 Ladder Structured Text

Operations Angle

  • Clear process logic supports productivity and reduces downtime from sequence errors.
  • Good interlocks reduce risk of equipment damage and safety incidents.
  • Simulation helps validate logic before real deployment (cost/time saving).
