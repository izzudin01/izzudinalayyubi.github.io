Smart Energy Meter (ESP32 + PZEM-004T)

IoT energy monitoring system: collect electrical parameters, publish to a web dashboard, and provide alerts for abnormal usage.

Objective

  • Measure voltage, current, power, and energy usage.
  • Enable remote monitoring via web UI.
  • Trigger alerts / relay action when usage exceeds thresholds.

System Overview

  • Sensor: PZEM-004T for measurement.
  • Controller: ESP32 (firmware in C++).
  • UI: lightweight web page (HTML/CSS/JS).

My Contribution

  • Integrated sensor reading + validation logic.
  • Built web interface for real-time monitoring (simple and responsive).
  • Implemented alert/notification logic and actuator control (relay) for abnormal conditions.
  • Documented wiring and basic safety considerations (isolation, fuse/MCB concept).

Tools & Tech

ESP32 C++ PZEM-004T HTML/CSS/JS Basic IoT

Why It Matters (Operations Angle)

  • Creates visibility of energy usage (supports cost awareness).
  • Enables early detection of abnormal consumption.
  • Practical foundation for preventive maintenance and energy-saving initiatives.
