P–N Diode Fabrication (Maskless Photolithography)

Semiconductor fabrication learning project: understanding process flow, critical parameters, and documentation discipline.

Objective

  • Understand end-to-end process flow for building a basic diode structure.
  • Identify critical steps and risks (alignment, exposure, etching, contamination).
  • Document process assumptions and expected device behavior.

Scope

  • Maskless patterning concept (laser-based).
  • Basic lithography → etch → dope/deposit flow understanding.
  • Learning-focused documentation rather than production-grade fabrication.

Approach

  • Mapped fabrication steps into a clear process sequence and rationale.
  • Noted key controls: cleaning, exposure/etch parameters, and inspection.
  • Connected fabrication steps to expected diode electrical behavior (qualitative).
