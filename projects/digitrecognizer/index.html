MNIST Digit Recognizer

Machine learning learning project: handwritten digit classification (0–9) using a simple neural network and structured evaluation.

Objective

  • Train a model to classify handwritten digits accurately.
  • Understand core ML workflow: preprocessing → training → evaluation.
  • Document performance and improvement opportunities.

What It Demonstrates

  • Structured experimentation mindset.
  • Basic model evaluation and error thinking.
  • Ability to learn new tools and methods quickly.

Approach

  • Preprocessed input images into a consistent numeric representation.
  • Trained a simple neural network (forward + backprop).
  • Evaluated accuracy and reviewed common misclassifications.

Note: This project is listed as “learning-focused” (supporting skill), not the main focus for UT MT.
